Trouble Viewing the Slides?

These slides are best viewed with Internet Explorer or Netscape browsers. If text letters run together, try clicking the "View" tab on your browser, then "Text Size," then select a smaller - or the smallest - size.

Another option to try is the PowerPoint Viewer available via free download from www.microsoft.com, search on "PowerPoint"

If none of these work, there are text versions of each presentation available in the site. Search the site on the main presentation topic to locate them. If you still have problems, please contact me.
